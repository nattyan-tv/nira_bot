# NIRA Bot
痒い所に手が届きそうで届かないDiscord用BOT

# Installation
## 必要なもの
- Python3.8以上

## セットアップ
### 方法1
1. 同梱の`setup.py`を実行します。(にらBOTを動かすPython/ユーザーで実行してください。)
2. 画面の指示の通りにします。

### 方法2
1. `pip install -r requirements.txt`でモジュールをインストールします。
2. `setting_temp.json`を参考にして、`setting.json`にTOKENなどの必要な設定を書き込みます。

## 起動
`nira.py`を実行します。
